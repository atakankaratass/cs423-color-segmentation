# Claude Instructions

This repository has mandatory contribution and validation rules.

Before making changes, read:

1. `CONTRIBUTING.md`
2. `AGENTS.md`
3. `docs/ai-tooling.md`

Required behavior:

- inspect repository state before editing
- follow the repository's mandatory AI workflow even if no plugin system is available
- make minimal, targeted changes
- add or update tests for behavioral changes
- do not bypass hooks, tests, lint, or CI requirements
- do not claim completion before local validation passes
- do not duplicate rules from the authoritative documents

Authoritative sources:

- contributor policy: `CONTRIBUTING.md`
- agent policy: `AGENTS.md`
